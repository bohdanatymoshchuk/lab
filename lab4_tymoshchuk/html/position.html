Демонстрація властивості position

static (значення за замовчуванням)

static

Елементи з position: static розташовуються у нормальному потоці документа.

relative

relative

Елемент з position: relative зміщується відносно свого нормального положення.

absolute

absolute

Елемент з position: absolute позиціонується відносно найближчого позиціонованого предка.

fixed

fixed

Елемент з position: fixed завжди залишається на одному місці навіть при прокручуванні сторінки.

sticky

sticky

Елемент з position: sticky поводиться як relative, доки не досягне заданого положення, після чого "прилипає" до цього положення.
