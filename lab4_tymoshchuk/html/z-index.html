Демонстрація властивості z-index

Властивість z-index визначає порядок накладання елементів один на одного. Елемент з більшим значенням z-index буде поверх елементів з меншими значеннями.

Зростаючий z-index (1, 2, 3)

z-index: 1
z-index: 2
z-index: 3

Спадаючий z-index (3, 2, 1)

z-index: 3
z-index: 2
z-index: 1

Для роботи z-index елемент повинен мати position відмінний від static (наприклад, relative, absolute або fixed).
